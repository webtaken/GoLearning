Nuestra Solución


estela es un clúster elástico de web scraping de código abierto que se ejecuta en Kubernetes. Proporciona mecanismos para implementar, ejecutar y escalar spiders de web scraping a través de una API REST y una interfaz web.

Los clientes y usuarios interesados en el web scraping pueden orquestar y ejecutar sus spiders (por ejemplo, arañas Scrapy). estela busca ser versátil, hasta el punto de que pueda implementarlo en su hogar.
